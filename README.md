## 커맨드 설명
L/l button : line mode 화면에 도트를 찍어서 그림 또는 글자를 적을수 있는 모드

T/t button : text mode 화면에 텍스트를 찍어서 원하는 글자를 보낼 수 있는 모드

I/i button : image mode 화면에 이미지를 띄우는 모드

Z/z button : On/Off

C/c button : 색깔을 랜덤으로 설정한다.

마우스 클릭 : 화면에 그려진 모든 요소들 삭제
